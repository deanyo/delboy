# 🍸 Del Boy's Drinks

**The definitive list of every drink ordered, mentioned or served in Only Fools and Horses (1981-2003).**

🔗 **[dnyo.co.uk/delboy](https://dnyo.co.uk/delboy/)**

## What is this?

There are dozens of "Del Boy drinks" lists online, but most of them are wrong. They mix up episodes, include drinks that were never actually ordered, and repeat fan myths as fact.

This project answers one question: *what did Del actually drink?*

To find out, we read every script from all 64 episodes, cross-referenced against DVD subtitles, and verified against broadcast footage.

The result is **44 confirmed drinks** — the most complete and accurate list anywhere online.

## Features

- 📋 **Searchable list** — filter by drink, episode, or who ordered it
- 🎰 **Spinning wheel** — random drink selector
- 🧪 **Drink generator** — AI-style Del Boy drink inventor using patterns from the show
- 📖 **Methodology page** — how we verified every entry
- 🔌 **Machine-readable API** — full dataset with drink types, locations, episode refs

## The Data

- **44 confirmed drinks** verified against scripts and broadcast
- **69 script files** analysed
- **31 DVD subtitle tracks** cross-referenced
- **1 open bounty** — Creme de Menthe and Tizer (if you can prove it exists, [email us](mailto:delboy@dnyo.co.uk))

### Key discoveries

- **Baileys and Cherryade** — script says Manhattan, but David Jason ad-libbed it on camera (Heroes and Villains)
- **Harvey Wallbanger** — I was convinced this was a fan hallucination. I was wrong — confirmed via DVD subtitles (Modern Men)
- **Irish Coffee** — "Del had a mutton Madras and four Irish coffees" (Strangers on the Shore)
- **Brandy and Andrews** — Del stirs liver salts into his brandy (Sickness and Wealth)
